# my-app

> 此项目主要为练习移动端响应式布局所建，并非实现微信的全部功能，数据来源于node模拟
>
> 项目需要安装mongodb客户端，关于Windows下mongodb的安装与启动，此处不多赘述
>
> #### 项目拷贝
>
> git clone  https://github.com/xiao11lang/Vue-WeChat.git
>
> #### 依赖安装
>
> cd Vue-WeChat
>
> npm install 
>
> #### 启动
>
> npm run dev 
>
> #### 服务端
>
> cd server
>
> node index.js
>
>
